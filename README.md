# Jobster

A fullstack app with lots of features.

Live Site URL: [jobster-api](https://node-jobster.onrender.com/)

## Built with

- Node.js
- React
- mongoDB


## Features:

- login/register
- JWT token
- add, delete, update your jobs
- personal profile
- protected data
